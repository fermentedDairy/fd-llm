# fd-llm
A collection of Tools, Agents and Skills for agentic code generation. This repo is focussed on the OpenCode agent, Agents and Skilla are comaptible with Claude Code and Copilot CLI. Tools are not.

Skills, tools and agents are organised into common usecases, with skills and tool adjacents to the agents that use them in the following strucutre:

```
Repository root    
│
└───use case 
│   └───agent
│       │   AGENT.md
│   └───skill
│       │   SKILL.md
│   └───tool
│       │   tool_file.ts
│       │   ...

```
